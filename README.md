# 瑞利 / 汤姆逊散射估算器

直接打开 `index.html` 即可使用，无需安装软件或联网。

计算器根据脉冲能量、脉宽、波长、气体温压、电子密度、收集长度、立体角、散射角、偏振条件和仪器响应参数，估算：

- 脉冲期间的瑞利散射功率
- 每脉冲汤姆逊散射光子数
- 脉冲期间的汤姆逊散射功率
- 按光路透过率 μ、量子效率 η、MCP 增益 G 折算的仪器等效计数

## 模型

```text
P_R = (E_L / tau) * (p / k_B T) * L * (d sigma / d Omega) * Delta Omega

P_T = (E_L / tau) * n_e * L * (d sigma_T / d Omega) * Delta Omega

N_T = P_T * tau * wavelength / (h * c)

N_T_inst = N_T * mu * eta * G
```

内置气体采用 532 nm、90 度、垂直偏振参考微分截面，并按 `(532 / wavelength)^4` 缩放。角分布采用理想偶极散射近似。

该工具用于实验设计和量级估算。绝对标定还应考虑有效收集长度随位置的变化、光学透过率、探测器响应、分子退偏振、滤光片带宽以及气溶胶米氏散射。
