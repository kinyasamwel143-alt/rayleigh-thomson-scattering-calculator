计算模型

瑞利散射功率
PR = (EL / τ) · ng · L · (dσR/dΩ) · ΔΩ

汤姆逊总光子数
NT = PT · τ · λ / (hc)

汤姆逊散射功率
PT = (EL / τ) · ne · L · (dσT/dΩ) · ΔΩ

仪器内汤姆逊等效计数
NT,inst = NT · μ · η · G

微分截面
dσT/dΩ = re²F(θ, pol),   dσR/dΩ = σR,532(532/λ)⁴F(θ, pol)

气体数密度
ng = p / (kBT)

其中 re = 2.8179 × 10⁻¹⁵ m，μ为全光路透过率，η为量子效率，G为 MCP 增益。按 μ·η·G 计算的仪器项是 MCP 等效计数；若只统计到达仪器的光子数，则只乘 μ。偏振角分布 F(θ, pol) 使用偶极散射近似；忽略多次散射、吸收、气溶胶米氏散射以及弱色散修正。

适合方案设计和量级估算。用于绝对标定时，应使用实测系统透过率、有效收集长度及对应波长的精确截面。
